## 🎯 猜数字

```
┌─────────────────────────────────┐
│      🎯  猜数字  🎯   🦔       │
│                                 │
│   剩余次数: 8   已猜: 2        │
│                                 │
│   请猜 1 到 100 之间的数字       │
│   ━━━━━━━━━━━━━━━━━━━ 20%      │
│                                 │
│   🦔 小刺猬提示：试试中间的数字！│
│                                 │
│   ⬇️ 太小了！再大一点！        │
│                                 │
│   📜 猜测历史:  [25↓] [50↓]    │
└─────────────────────────────────┘
```

**游戏规则：**
- 🦔 小刺猬会想一个 1-100 的随机数字
- 你有 10 次机会猜测
- 每次猜测后，小刺猬会提示"太大"或"太小"
- 用最少的次数猜中数字，刷新你的最佳记录！

**小技巧：** 使用二分法可以快速缩小范围哦~

## ✊ 石头剪刀布

```

│   胜利: 3   失败: 2   平局: 1  │

## 🧮 24点计算

```
┌─────────────────────────────────┐
│     🧮 24点计算 🦔              │
│                                 │
│   得分: 30   回合: 4   连胜: 3 │
│                                 │
│   🦔 正在思考中...              │
│                                 │
│   目标: 24                     │
│                                 │
│   ┌──┐ ┌──┐ ┌──┐ ┌──┐          │
│   │ 3│ │ 7│ │ 8│ │ 1│          │
│   └──┘ └──┘ └──┘ └──┘          │

│   表达式: (8-1)×3+7            │
│   ┌─────────────────────────┐  │

## 🎲 幸运色子

```
┌─────────────────────────────────┐
│     🎲 幸运色子 🦔              │
│                                 │
│   金币: 150   胜利: 5  连胜: 3 │
│                                 │
│   🦔 今天运气不错呢！           │
│                                 │
│   ┌────────┐    ┌────────┐     │
│   │   5    │ VS │   3    │     │
│   │你的色子│    │ 🦔色子  │     │
│   └────────┘    └────────┘     │
│                                 │
│   🎉 你赢了！+40 金币           │
│                                 │
│   选择投注: [10] [20] [50] [100]│
│                                 │
│        [🎲 投掷色子]            │
└─────────────────────────────────┘
```

**游戏规则：**
- 同时投掷两枚色子进行比较
- 选择投注金额（从你的金币中扣除）
- 你的点数 > 🦔 小刺猬点数 = 胜利，获得双倍投注
- 你的点数 < 🦔 小刺猬点数 = 失败，损失投注金额
- 点数相同 = 平局，退还投注金额
- 连胜可获得额外金币奖励！

## ✨ 特色功能

- 🦔 **可爱刺猬** - 底部有一群蹦蹦跳跳的小刺猬陪伴你
- 🌟 **入场特效** - 选择游戏时有星星爆发动画
- ⚡ **冲刺效果** - 页面切换时的速度线特效
- 🎨 **精美UI** - 现代化渐变设计和流畅动画
- 📱 **响应式** - 支持手机、平板、电脑全平台
- ⚡ **无需安装** - 纯浏览器运行，即开即玩
- 💾 **数据保存** - 本地记录游戏进度和最佳成绩
- 🎯 **多种难度** - 满足不同水平的玩家需求
